# UNGROUP

Separates a grouped object into individual objects. Use UNGROUP to
separate a group of objects so that you can format, move, or size one of
the objects.

If the selection is not a grouped object, UNGROUP returns FALSE.

**Syntax**

**UNGROUP**( )
