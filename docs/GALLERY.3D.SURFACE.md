# GALLERY.3D.SURFACE

Changes the active chart to a 3-D surface chart.

**Syntax**

**GALLERY.3D.SURFACE**(**type\_num**)

**GALLERY.3D.SURFACE**?(type\_num)

Type\_num&nbsp;&nbsp;&nbsp;&nbsp;is the number of the 3-D Surface format
that you want to apply to the chart.
