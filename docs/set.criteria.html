SET.CRITERIA macro

Equivalent to clicking the Set Criteria command on the Data menu in Microsoft Excel version 4.0. Defines the name Criteria for the selected range on a sheet or macro sheet.

Syntax

SET.CRITERIA( )

Related Functions
