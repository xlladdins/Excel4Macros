PIVOT.REFRESH macro

Refreshes a PivotTable report.

Syntax

PIVOT.REFRESH(name)

Name    is the name of the PivotTable report the user would like to refresh with current data. If name is omitted, Microsoft Excel will use the PivotTable report containing the active cell.

Remarks

  • If the function is successful, it returns TRUE; otherwise, it returns the #VALUE! error value.

  • If name is not a valid PivotTable name, then the #VALUE! error value is returned.
