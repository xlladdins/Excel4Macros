SIZE macro

Equivalent to clicking the Size command on the Control menu in Microsoft Excel for Windows version 3.0 or earlier or to changing the size of a window by dragging its border. In Microsoft Excel for the Macintosh version 3.0 or earlier, equivalent to changing the size of a window by dragging its size box. This function is included only for macro compatibility and will be converted to WINDOW.SIZE when you open older macro sheets. For more information, see WINDOW.SIZE.

Syntax

SIZE(width,height,window_text)

SIZE?(width,height,window_text)

Related Function

WINDOW.SIZE   Changes the size of the active window
