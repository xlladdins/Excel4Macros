CLOSE.ALL macro

Equivalent to clicking the Close All command on the File menu. The Close All command appears when you hold down SHIFT while selecting the File menu. Closes all protected and unprotected windows and all hidden windows. If unsaved changes have been made to the workbook in one or more windows, a message is displayed asking if you want to save each workbook.

Syntax

CLOSE.ALL( )

Related Functions

CLOSE   Closes the active window

FILE.CLOSE   Closes the active workbook

QUIT   Ends a Microsoft Excel session

SAVE   Saves the active workbook
