SELECT.CHART macro

Equivalent to the Select Chart command on the Chart menu in Microsoft Excel version 4.0. This function is equivalent to using the third form of SELECT with "Chart" as the item_text argument.

Syntax

SELECT.CHART( )

Remarks

This function is included for compatibility with macros written with Microsoft Excel for the Macintosh version 1.5 or earlier.

Related Function

SELECT   Selects a chart object
