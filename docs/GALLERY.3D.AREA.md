# GALLERY.3D.AREA

Changes the format of the active chart to a 3-D area chart.

**Syntax**

**GALLERY.3D.AREA**(**type\_num**)

**GALLERY.3D.AREA**?(type\_num)

Type\_num&nbsp;&nbsp;&nbsp;&nbsp;is the number of the 3-D Area format
that you want to apply to the chart.
